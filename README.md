## Installation

```julia
using Pkg
Pkg.add("https://github.com/vidhyasaharan/VectorDataUtils.jl.git")
```
VectorDataUtils is supported on Julia 1.6 and later. It is tested on Julia 1.6, Julia 1.7 and nightly.
